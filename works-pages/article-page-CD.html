Carpe Diem

2023 Web application

This is a free application for online recognition of the images you download.
You can use the application absolutely free.
This application can recognize objects in photos in popular formats: JPG, PNG.

A neural network based on the cifar-10 dataset, with the help of which we examine the picture for belonging to one of 10 classes, such as:

  • airplane
  • car
  • birds
  • cat
  • deer
  • dog
  • frog
  • horse
  • ship
  • truck

The application written in:

  • backend - Django
  • frontend - HTML, CSS, Bootstrap
  • neural network model - ResNet 50

Command to start:

  • python manage.py runserver
